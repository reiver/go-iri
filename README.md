# go-iri

Package **iri** provides tools for working with **IRI** (**internationalized Resource Identifier**), for the Go programming language.

Note that package **iri** is intentionally similar to the Go built-in package `"net/url"`.

## Documention

Online documentation, which includes examples, can be found at: http://godoc.org/github.com/reiver/go-iri

[![GoDoc](https://godoc.org/github.com/reiver/go-iri?status.svg)](https://godoc.org/github.com/reiver/go-iri)

## Import

To import package **iri** use `import` code like the following:
```
import "github.com/reiver/go-iri"
```

## Installation

To install package **iri** do the following:
```
GOPROXY=direct go get github.com/reiver/go-iri
```
